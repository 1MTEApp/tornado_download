TORNADO

Tornado - программа для шифровки и расшифровки текста и не только! Поиск по интернету прямо в программе!

Tornado работает полностью на python, но с использованием библиотек:

Time - предоставляет различные функции для работы со временем.

Cryptography(pip install cryptography) - библиотека для шифровки.

Art(pip install art) - предоставляет возможности для создания ASCII-текста из обычного текста.

bs4(pip install beautifulsoup4) - парсинг HTML и XML документов в Python.

requests(pip install requests) - выполнения HTTP-запросов на языке Python.
